readme.txt
==========

This directory contains snake files for use with vivaxGEN ngs-pipeline.
The following are brief description of those files:

* ssf_* (single sample flow) are snake files containing rules that can only
  handle a single sample for each snakemake process

* msf_* (multi sample flow) are snake files containing rules that can handle
  multiple samples in a single snakemake process.
